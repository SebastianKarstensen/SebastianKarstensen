Autobook

The Autobook application was the result of a group project for the course "Software Implementation" at Syracuse University. The app allows users to plan social media updates on Twitter and Facebook, as well as scheduling text messages. You are also able to import contact informations such as Twitter handles and phone numbers.

AtSea

As part of my bachelor's project in procedural content generation, the game "AtSea" was developed. The project had been based around generating a weather system, and the game was used to test whether or not the system could implemented as a game mechanic. A video about the project can be found here.
